_OpenMarch: On The Move_ is our upcoming mobile app for distributing shows you write with OpenMarch and other tools. Public testing helps us find issues before a wider release.

## Demo login

After installing On The Move, use these demo credentials to get started:

- Ensemble code: `OTMDEMO`
- Password: `ONTHEMOVE`

## Getting access

Testing is currently available through TestFlight on iOS. Android testing is not yet available and is coming soon.

### iOS testing

Use TestFlight to install the iOS beta:

1. Install [TestFlight](https://apps.apple.com/us/app/testflight/id899247664) from the App Store if you do not already have it.
1. Open the [On The Move TestFlight invite](https://testflight.apple.com/join/WRWf9wmN).
1. Accept the invite and install On The Move from TestFlight.

### Android testing

Android testing is not yet available. We are working on bringing On The Move to Android and will open public testing through Google Play when it is ready.

If you are on Android, check back here for updates.

## What to test

After installing the app, try using On The Move in the same way you would during rehearsal, planning, or quick review sessions.

- Log into an ensemble
- Test playback, page navigation, and count stepping
- Note anything confusing, slow, or unexpected
- Try to break things

Check out our [full testing guide](https://docs.google.com/document/d/1kiup_kpB4ALTM3J2KHcGdUny2FqdekMKiqfUP_lcjK8/edit?usp=sharing).
